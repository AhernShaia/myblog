Harbor

概述:

是由 VMWare 開源的容器鏡像倉庫,基於Docker registry做的擴展。

配置需求:

  • CPU 2 核/記憶體 4G/硬碟空間 40GB
  • 建議配置: 4 核/記憶體 8G/硬碟空間 160GB
  • Docker CE 17.06 以上
  • Docker Compose 1.18 以上

安裝方式:

Docker Hub 下載 Harbor 相關鏡像。

安裝包,包含相關部署鏡像,佔用空間較大。

部署 HTTP

1、安裝 Docker 和 Docker Compose

下載 docker-compose 並安裝

2、下載 harbor 安裝、啟用

登入 harbor

⚠️ 登入成功

推送鏡像

點選 項目名 → 推送鏡像的 Docker 命令 可查看推送命令

部署 HTTPS

  1. 生成 SSL 證書

  2. Harbor 啟用 HTTPS

  3. 重新配置、部署 Harbor

    ⚠️ 查看憑證是我們生成的證書

客戶端訪問配置

從其他主機 docker login 登入至服務端的 harbor

‼️ 由于harbor未配置https，需要在docker配置可信任。
⚠️ 證書簽名問題

將證書複製到客戶端

複製拉取鏡像命令

⚠️ 客戶端成功拉取到鏡像

主從複製

目的是解決高可用問題

  • 主備
    簡易,主掛了之後會切會備用 Harbor
    同時只提供一台服務
    適合少量鏡像傳輸

  • 雙主複製
    雙向配置複製
    同時兩台服務運行
    需要負載均衡

  • 一主多從
    多個從同步主配置
    適合多區域業務(跨機房),大量鏡像傳輸需求

主備環境搭建

  • 兩台虛擬機: 1 台主服務 、1 台客戶端
  • harbor
  • docker-compose
