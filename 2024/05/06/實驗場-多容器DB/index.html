實驗場-多容器DB

這個實驗主要用於無法在 macOS 上啟動mssql而衍生出的實驗環境。

使用工具
  • DBeaver
  • docker

1. 啟動container

2. DBeaver 連接 mssql

a. 選擇SQL Server點選完成

b. 輸入資料庫的 IP，登入的用戶及密碼，點選完成後，第一次連接會出現安裝套件的視窗。

c. 完成

3. 建立資料

點選文件展開下拉式選單，選擇導入

a-1. 導入腳本(SQL)


a-2 選擇腳本所在的目錄，點選完成


a-2 對要執行的腳本右鍵，選擇打開資源

4. 執行腳本(SQL)

點擊執行 SQL 腳本
